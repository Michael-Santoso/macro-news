# AGENTS-FRONTEND.md

Project:
Macro Tracker frontend for hackathon demo.

Stack:

- Next.js
- TypeScript
- modern component-based frontend

Design Intent:

- Bubble visualization is only the hero/landing section
- The rest of the dashboard should be clean, structured, and evidence-based
- UI should feel fintech-like, modern, and readable

Rules:

- Do not redesign the product concept
- Do not build a chaotic graph-heavy UI
- Prefer stable layouts and clear interactions
- Keep components modular
- Keep styling consistent
- Avoid unnecessary dependencies unless clearly useful
- Before coding, list files you will inspect and modify
- After coding, summarize files changed and what was implemented

Architecture Guidance:

- Shared types should be centralized
- Shared API logic should be centralized
- Keep page state lifting simple and explicit
- Avoid duplicating backend response interfaces across components

Priority:

1. Demo clarity
2. Readability
3. Visual impact
4. Responsiveness
5. Nice-to-have animations
